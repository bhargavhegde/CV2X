Commsignia V2X Remote Python SDK
======================================

Commsignia V2X Remote Python SDK is a python package to access low level
functionality of the Commsignia V2X stack. It handles connecting to the stack
through a proprietary TCP protocol and sends RPC commands.

# Components

- built distribution is the _.whl_ file
- examples in `examples`

# Installing

- the package requires python 3.7 or newer version
- for installing the package you need _pip3 install $whl_dist_ (whl_dist is the built distribution)

### CRH Note
Better to use a virtual environment. Specifically, on Mac os (M1 chip) 
'''
    python3 -m venv path/to/venv
    source path/to/venv/bin/activate
    python3 -m pip install xyz
'''

# Running the examples

- the examples can be run with _python3 some_example.py_
