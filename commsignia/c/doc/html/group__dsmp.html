typedef void(* cms_dsmp_rx_notify_f) (cms_psid_t aid, const cms_dsmp_rx_notif_data_t *notif, cms_buffer_view_t msg, void *ctx)

Received DSMP notify callback function type.

When a notification is sent to a subscribed client the following typed function callback is called. This callback function shall be registered during cms_dsmp_rx_subscibe function call.

Parameters
aidAID of the received message
notifNotification metadata
msgRaw payload buffer
ctxUser context, registered during subscribe

Subscribe for received DSMP messages.

Upon successful subscription, when a DSMP message is received with a matching AID, the ITS software stack sends a notification to the subscribed client with some metadata and the raw message.

Parameters
sessionClient session
aidAID to subscribe for
callback_fNotification callback function
ctxUser context
[out]subs_idSubscription ID output
Returns
true in case of error

◆ cms_dsmp_rx_unsubscribe()

bool cms_dsmp_rx_unsubscribe ( const cms_session_t *  session,
cms_subs_id_t  subs_id 
)

Unsubscribe from received DSMP messages.

Upon successful unsubscription, when a DSMP message is received with a matching AID, the ITS software stack does not send a notification to the subscribed client any more. Messages with different subscribed AID will still be sent as a notification.

Parameters
sessionClient session
subs_idSubscription ID, retrieved by cms_dsmp_rx_subscribe
Returns
true in case of error
