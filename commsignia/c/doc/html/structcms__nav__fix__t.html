bool cms_nav_fix_t::is_valid

Flag to indicate whether the NAV fix is valid.

Note
If this flag is true, all other fields will be valid, recent and consistent. If the flag is false, either all fields will be N/A, or they will contain the most recent consistent data, but it can be very old.
