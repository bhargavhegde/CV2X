Transmitted DSMP notify callback function type.

When a notification is sent to a subscribed client the following typed function callback is called. This callback function shall be registered during cms_dsmp_tx_subscribe function call.

Parameters
aidAID of the received message
notifNotification metadata
msgRaw payload buffer
ctxUser context, registered during subscribe
