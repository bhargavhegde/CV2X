Detailed Description

Client session.

A client session provides a handle to a host-client connection. It is used as an argument in all API functions except cms_get_session() and cms_get_session_for_id().

Note
Always create / initialize a session object with cms_get_session() or cms_get_session_for_id(). User modification of the session object is not allowed.
