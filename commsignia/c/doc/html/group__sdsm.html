typedef uint16_t cms_sdsm_object_id_t

ID parameter used in the SDSM module.

Warning
This is an experimental API. It's name, behaviors and parameters may change and it may be removed without prior notice.

Provides a unique identifier that is assigned to each detected object. The assigned ObjectID is maintained for a detected object as long as the object is detected.
