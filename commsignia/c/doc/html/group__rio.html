bool cms_rio_rx_unsubscribe ( const cms_session_t *  session,
cms_subs_id_t  subs_id 
)

Unsubscribe from received radio messages.

Upon successful unsubscription, when a radio message is received with a matching packet type, the ITS software stack does not send a notification to the subscribed client any more. Messages with different subscribed packet type will still be sent as a notification.

sessionClient session

◆ CMS_RIO_SUBSCRIBE_ALL

const cms_rio_subscribed_packet_t CMS_RIO_SUBSCRIBE_ALL = 0UL
static

Wildcard value for the radio receive subscribe function's type parameter.

When this value is used as a key for the radio receive subscribe function, all received packets will trigger a notification.
