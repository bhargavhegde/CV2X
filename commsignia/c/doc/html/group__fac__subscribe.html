Facility unsubscribe.

Upon successful unsubscription, when a facility message is received with a matching type, the ITS software stack does not send a notification to the subscribed client any more. Messages with different subscribed type will still be sent as a notification.

Parameters
sessionClient session
subs_idSubscription ID, retrieved by cms_fac_subscribe
Returns
true in case of error

const cms_fac_subscribed_msg_t CMS_FAC_SUBSCRIBE_ALL = 0UL
static

Wildcard value for the facility subscribe function's type parameter.

When this value is used as a key for the facility subscribe function, all received facility messages will trigger a notification.
