### Examples

Examples of C-V2X data recording and WSMP customized message sending. 
Generated based on examples provided by Commsignia.

Please refer to [Commsignia](https://buffalo.box.com/s/uoe7r7khazz3pa8peidn9etjti41ds6p) folder for details 
- C-V2X protocol and standards
- Commsignia C-V2X devices documentation 
- SDKs to setup Commsignia APIs. 

Please refer to [Recorder](Recorder/README.md) for details on how to use the Recorder module with Commsignia devices.
